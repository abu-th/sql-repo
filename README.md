## 📌 Project Overview

This project was completed as part of my **Data Technician Bootcamp**, focusing on developing practical SQL skills using MySQL.

The work involved querying relational databases to explore sales, customer, product, and operational data, with the goal of extracting meaningful business insights.

It reflects real-world SQL usage commonly applied in reporting, analytics, and database management roles.

---

## 🎯 Project Objectives

- Query structured relational databases effectively  
- Extract and filter relevant business data  
- Combine tables using joins for deeper analysis  
- Summarise datasets using aggregation techniques  
- Generate insights for business and operational use cases  

---

## 🧠 Key SQL Skills Applied

### 🔍 SELECT Queries
Used to retrieve data from tables such as customers, products, orders, and cities.

---

### 🎯 Filtering with WHERE
Applied conditions to focus on specific data segments, including:

- High-value products  
- Customers from selected regions  
- Orders within specific time periods  
- Cities with defined population thresholds  

---

### 🔢 Sorting with ORDER BY
Organised results to support analysis, such as:

- Recent orders first  
- Products by price  
- Cities by population size  

---

### 📊 Aggregation with GROUP BY
Summarised datasets to identify patterns, including:

- Product counts per category  
- Total quantities ordered  
- Average values across regions  

---

### 🔗 Table Joins
Used multiple join types (INNER and LEFT JOIN) to connect related datasets:

- Products linked with suppliers  
- Orders connected to customers and employees  
- Categories matched with product lists  

These joins helped build a more complete view of business operations.

---

## 📈 Example Insights Generated

- Identified top-performing products based on order volume  
- Analysed customer distribution across different countries  
- Reviewed city-level demographic patterns for reporting insights  
- Mapped suppliers to products for supply chain visibility  
- Generated time-based summaries for order tracking and analysis
